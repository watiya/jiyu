Starzip 검색만의 특이점 — 일반 DEX는 토큰 심볼(BTC, ETH)만 검색합니다. Starzip은 상점명·지역·업종·나라·도시라는 실세계 맥락을 검색 축으로 추가합니다. "홍대 카페", "도쿄 피트니스" 같은 자연어 쿼리를 지원하는 것이 크립토 DEX 중 유일한 차별화 포인트입니다.
1

검색 진입

검색창 클릭 또는 Cmd+K 단축키. 마켓 어느 화면에서도 접근 가능 (글로벌 트리거). 진입 즉시 드롭다운 오픈 — 최근 검색어 + 업종/지역 빠른 탐색 표시.

트리거: 클릭 · Cmd+K · 모바일 탭
2

입력 전 — 빠른 탐색 제공

최근 검색어 5개 + 업종 카테고리 칩 + 주요 지역 목록 표시. 검색어 없이도 카테고리·지역으로 바로 진입 가능.

표시 조건: 검색창 포커스 & 입력값 없음
3

입력 중 — 실시간 자동완성

1자 이상 입력 시 디바운스 150ms 후 API 호출. 상점·토큰 / 글로벌 같은 업종 / 업종 / 지역 4개 섹션으로 결과 표시. 입력 글자 하이라이트.

트리거: keyup (디바운스 150ms)
4a

결과 선택 — 상점 직접 클릭

드롭다운 내 상점 항목 클릭 → /trade/CAFE 거래 화면으로 직접 이동. 가장 빠른 경로.

결과: /trade/[TOKEN] 페이지 이동
4b

결과 선택 — 업종/지역 클릭

업종 또는 지역 항목 클릭 → /markets?category=카페 또는 ?region=성수동으로 이동. 사이드바 필터 자동 동기화.

결과: /markets?category= 또는 ?region= 이동
4c

결과 선택 — Enter / 전체 결과 보기

Enter 입력 또는 전체 결과 보기 클릭 → /markets?q=[검색어] 이동. URL에 반영되어 공유 가능, 뒤로가기 복귀 가능.

결과: /markets?q=[검색어] 이동
5

결과 없음 처리

매칭 결과 0개 → 유사 업종 칩 제안 + "상점 제안하기" CTA. Esc 또는 X 클릭으로 초기화.

조건: API 응답 count === 0
