고유

C. 사전 수요 예측 (런치 D-3 이전)
C1
사전 등록 패널 NEW
등록자 수 + 예상 달성률 + 15% 기준선 + 원클릭 등록 버튼
필수
고유
수치 표시
  • 사전 등록자 수 (실시간)
  • 예상 참여금액 합산
  • 소프트캡 달성 전망 %
  • 15% 기준선 (소프트캡 50%) 마커
CTA
  • "참여 의향 등록" 파란 버튼
  • 등록 후 → 체크 + "알림 수신 중"
  • 지갑 미연결 시 → "지갑 연결하기"
  • 등록 취소 링크 (회색 텍스트)
경고 배너
  • 15% 미달 예상 시 노란 경고 배너
  • "현재 페이스로는 런치가 취소될 수 있어요"
  • 취소 가능 (X 버튼)
C2
카운트다운 타이머 NEW
런치 오픈까지 D-day·시·분·초 실시간 카운트다운
필수
고유
표시 형식
  • D-3 / 02:45:30 형식
  • 일·시·분·초 4칸 그리드
  • 초 단위 실시간 갱신 (setInterval)
  • 0이 되면 → 공모 패널로 자동 전환
부가 정보
  • 공모 오픈 예정 날짜·시간 (KST)
  • STAR 스테이킹 부스트 안내 링크
C3
STAR 부스트 안내 패널 NEW
STAR 스테이킹 수량별 배정 배율 테이블 + 스테이킹 바로가기
필수
고유
배율 테이블
  • 0 STAR → 1.0x (기본)
  • 1,000 STAR → 1.2x
  • 5,000 STAR → 1.5x
  • 10,000 STAR → 2.0x
내 현황
  • 현재 스테이킹 수량 (지갑 연결 시)
  • 내 배율 강조 (파란 행)
  • "STAR 스테이킹 하러 가기" 버튼
  • 배율은 비례 배정에 곱해짐 안내
D. 공모 진행 (런치패드 오픈 후)
D1
공모 진행률 패널 NEW
달성액·달성률·소프트캡 마커·참여자 수·마감 카운트다운
필수
고유
진행률 바
  • 현재 달성액 / 목표금액
  • 달성률 % (실시간)
  • 소프트캡 30% 기준선 마커 (세로선)
  • 하드캡 100% 도달 시 색상 전환
수치 그리드
  • 총 참여금액 (USDT)
  • 참여자 수 (고유 지갑 수)
  • 1인당 평균 참여금액
  • 공모 마감까지 남은 시간
초과 청약 표시
  • 100% 초과 시 "초과 청약 중" 뱃지
  • 비례 배정 안내 토글
  • 내 예상 배정 비율 실시간 계산
D2
참여 입력 폼 NEW
USDT/USDC 입력 + 잔액 표시 + STAR 부스트 배율 + 예상 토큰 수량
필수
고유
입력 필드
  • USDT / USDC 선택 탭
  • 금액 입력 (숫자, monospace)
  • 잔액 표시 + MAX 버튼
  • 최소 참여금액 경고 (미달 시)
예상 수령량
  • 내 참여금 / 공모가 = 예상 토큰 수
  • STAR 부스트 배율 적용 후 가중치
  • 비례 배정 후 최소~최대 범위 표시
  • 미배정 금액 자동 환불 안내
OCT 희석 공시
  • OCT 발행량 (공모량의 10%)
  • 예상 희석률 ~9.1%
  • 경고 박스 (필수 사전 공시)
D3
참여 확인·실행 버튼 NEW
참여 버튼 → 확인 모달 → AA 서명 → 에스크로 입금 → 완료
필수
고유
버튼 상태
  • 기본: "참여하기" (파란 버튼)
  • 지갑 미연결: "지갑 연결하기"
  • 금액 미입력: disabled
  • 최소금액 미달: "최소 $10 이상 입력" 경고
  • 공모 마감: "참여 마감" (회색)
확인 모달
  • 참여금액 · 예상 토큰 · 수수료 요약
  • OCT 희석 재확인 체크박스
  • "에스크로에 안전하게 보관됩니다" 안내
  • 확인 → 에스크로 컨트랙트 전송
D4
내 참여 현황 NEW
내 참여금액·예상 배정·부스트 배율·취소 가능 여부 표시
필수
고유
현황 카드
  • 내 참여금액 (USDT)
  • 적용 STAR 부스트 배율
  • 가중 참여금 (참여금 × 배율)
  • 현재 예상 배정 비율 (실시간)
  • 예상 수령 토큰 수 (최솟값)
액션
  • 참여 취소 (공모 기간 중만 가능)
  • 추가 참여 (폼으로 스크롤)
  • 알림 설정 (마감 임박 알림)
D5
참여자 현황 리스트 NEW
최근 참여자 (익명화 지갑 주소) + 금액 + 시간 실시간 피드
높음
BN 참고
표시 정보
  • 지갑 단축 주소 (0x...1234)
  • 참여금액 (USDT)
  • 참여 시각 (N분 전)
  • 최근 20건 표시, 실시간 업데이트
집계 수치
  • 고유 참여자 수
  • 평균 참여금액
  • 최대 단일 참여금액
E. 배정·정산 (공모 종료 후)
E1
소프트캡 판정 결과 NEW
달성 → 진행 / 미달 → 전액 자동 환불 배너. 스마트 컨트랙트 자동 처리
필수
고유
달성 시
  • 초록 배너: "소프트캡 달성! 공모 진행"
  • 최종 달성액 · 참여자 수 표시
  • 배정 결과 확인 안내
  • DEX 상장 일정 공지
미달 시
  • 빨강 배너: "소프트캡 미달 · 전액 환불"
  • 에스크로 자동 환불 진행 중 표시
  • 예상 환불 완료 시각 (블록 확인 기준)
  • 온체인 해시 링크
자동 처리: 소프트캡 미달 시 에스크로 컨트랙트가 참여자 전원에게 원래 통화(USDT/USDC)로 자동 환불. CS 개입 없음.
E2
비례 배정 결과 카드 NEW
내 배정 토큰 수 + 환불금액 + STAR 부스트 최종 반영 결과
필수
고유
배정 결과
  • 최종 배정 토큰 수 (CAFE)
  • 배정 기준 금액 (USDT)
  • 미배정 환불금 (USDT)
  • 적용 STAR 부스트 배율
계산 내역
  • 내 가중 참여금 / 전체 가중 참여금 = 배정 비율
  • 배정 비율 × 공모 물량 = 배정 토큰
  • 온체인 기록 해시 링크
액션
  • "클레임하기" 버튼 (TGE 후 활성)
  • 클레임 전: 잠금 아이콘 + TGE 날짜
  • 환불금 자동 지갑 전송 완료 확인
E3
공모금 배분 공시 NEW
40% 즉시·10% 마켓메이킹·50% 베스팅 배분 시각화 + 온체인 해시
필수
고유
배분 시각화
  • 가로 바 차트 (40/10/50)
  • 즉시 지급 40% (파랑)
  • 마켓메이킹 10% (회색, 12개월 락업)
  • 베스팅 50% (녹색, 12개월)
금액 표시
  • 각 항목별 실제 USDT 금액
  • 즉시 지급: 사업주 수령 완료 표시
  • 베스팅: 월 해제액 계산 (~4.2%/월)
온체인 증거
  • 배분 트랜잭션 해시 3개
  • 블록 익스플로러 링크
  • "투명하게 기록됩니다" 안내
E4
미판매 토큰 소각 공시 NEW
공모 미판매 물량 소각 트랜잭션 표시. 총 공급량 감소 확인
필수
고유
소각 정보
  • 소각 수량 (공모 미판매분)
  • 소각 후 총 공급량
  • 소각 트랜잭션 해시
  • 소각 주소 (0x000…dead)
표시 시점
  • 공모 종료 즉시 자동 소각
  • 소각 완료 시 배지 표시
  • 블록 탐색기 링크 제공
Starzip 정책: 공모 미판매 토큰은 즉시 소각. 별도 Reserve 없음. 총 공급량이 실제 유통 가능량과 일치하도록 설계.
F. DEX 상장 전환
